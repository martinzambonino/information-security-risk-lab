# Security Policy

## Educational Scope

The tools, templates, and code contained in this repository are developed strictly for academic, research, and educational purposes. Risk assessment outputs and compliance mappings are based on fictional scenarios and must not be used as-is for real organizational decision-making.

## Reporting a Vulnerability

If you discover a security vulnerability within this project itself, please do not disclose it publicly.

Instead, open an issue labeled `security` or contact the repository owner directly. Given the educational nature of the project, reasonable effort will be made to remediate architectural or dependency vulnerabilities.

## Responsible Use

By cloning or downloading materials from this repository, you agree to:

1. Not present these materials as professional risk consulting deliverables.
2. Not use synthetic EITDP templates for real regulatory filings without proper legal review.
3. Assume full responsibility for compliance with all applicable local and international laws.
4. Acknowledge that risk matrices and calculations are simplified for educational purposes.
